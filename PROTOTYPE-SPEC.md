# 原型页面规格（给 Pencil MCP 用）

产品：推荐系统运营管理后台
设计风格：深色主题，参考 design-system/variant-view-1.html
- 背景：#151515(主区域) / #271418(侧边栏)
- 卡片：#222222，hover #2A2A2A
- 文字：白色 / rgba(255,255,255,0.45) / rgba(255,255,255,0.25)
- 主色：渐变 #4A4DFF→#E91E63→#FF9800
- 圆角：8-12px
- 字体：系统字体

全局布局：左侧固定导航栏 240px + 右侧主内容区

侧边栏导航项：
- 选品管理 > 商品选品池
- 排序策略 > 策略列表
- 策略组合
- 投放计划
- 效果监控
- 工具&诊断 > 全链路预览
